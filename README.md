# Kon Bnega GOAT? 🐐

An interactive pop culture quiz application with a beautiful pastel color palette, multiple game modes, and engaging features.

## Website:
https://quiz-app-phi-woad.vercel.app/

## Features

✨ **Interactive Quiz Experience**
- 100+ pop culture questions across 6 categories
- Multiple choice questions with instant feedback
- Time-limited questions with visual timer rings
- Score tracking and results summary

🎯 **Game Modes**
- **Classic Quiz**: Answer questions and track your score
- **Flashcard Mode**: Study questions in flashcard format
- **Leaderboard**: View top scores and compare performance

💪 **Power-Ups & Lifelines**
- 50/50 Lifeline: Eliminate two incorrect options
- Skip Question: Move to the next question
- Hint System: Get helpful clues

🎨 **Design**
- Elegant pastel color palette (lavender, peach, mint, sky blue, cream)
- Responsive dark theme UI
- Smooth animations and transitions
- Mobile-friendly interface

📊 **Features**
- Player name customization
- Real-time score display
- Category selection
- Question review after quiz
- Performance statistics

## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Styling with CSS variables, animations, and gradients
- **Vanilla JavaScript** - No dependencies, pure JS logic
- **Responsive Design** - Works on desktop and mobile devices

## How to Use

1. **Clone or download** this repository
2. **Open `index.html`** in your web browser
3. **Enter your name** to get started
4. **Select a category** to begin the quiz
5. **Answer questions** and use lifelines wisely
6. **View your results** and track your performance

## Features in Detail

### Categories
Choose from 6 diverse pop culture categories:
- Movies 🎬
- Music 🎵
- TV Shows 📺
- Gaming 🎮
- Celebrities 🌟
- Memes & Trends 😄

### Lifelines
Each lifeline can be used once:
- **50/50**: Removes 2 wrong answers
- **Skip**: Move to the next question
- **Hint**: Get a helpful clue

### Quiz Mechanics
- **Time Pressure**: Questions have a countdown timer
- **Instant Feedback**: See if your answer is correct immediately
- **Score Tracking**: Real-time score display
- **Review Option**: Review all questions after completing the quiz

## Performance Tips

- Use lifelines strategically
- Don't rush through questions
- Review your answers to learn from mistakes
- Try different categories to expand your knowledge
